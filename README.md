### 插件依赖

| 插件              | 目的         |
| ----------------- | ------------ |
| chalk             | 美化输入输出 |
| prompts           | 用户信息输入 |
| download-git-repo | 远程模板下载 |
| cac               | cli指令      |
| lodash.template   | 模板替换     |
| ora               | loading效果  |
| fast-glob         | 文件读取     |

### 使用方式

```tsx
rtc create <template>

rtc c <template>

rtc list <template>

rtc ls <template>
```
